## URL-Shortener
#### Программа представляет собой Rest API сервис для сокращения ссылок.

**Основные инструменты**

*Go	1.25.1	Основной язык разработки*

*PostgreSQL	17	Хранение пар "алиас → URL"*

*Docker Compose	Контейнеризация и оркестрация*

*Go Modules	-	Управление зависимостями*

#### Используемые библиотеки и их назначение

**import "github.com/joho/godotenv"**

Загрузка переменных окружения из файла .env для локальной разработки.

**import "github.com/ilyakaznacheev/cleanenv"**

Автоматическое чтение конфигурации из YAML-файлов и переменных окружения.

**chi — Маршрутизация HTTP-запросов**

Лёгкий, быстрый и совместимый с net/http роутер.

**chi/middleware — Базовые middleware**

*RequestID — генерирует уникальный ID для каждого запроса*

*Logger — базовое логирование*

*Recoverer — восстановление после паники (не падать, а вернуть 500)*

*URLFormat — поддержка форматов .json, .xml*

*Timeout — ограничение времени выполнения (60 сек)*

*BasicAuth — защита эндпоинтов логином/паролем*

**slog — Структурированное логирование** 

Стандартная библиотека Go для логирования с уровнями и атрибутами.
Разные уровни: Debug, Info, Error

*Структурированные поля: slog.String("key", "value")*

*Легко переключить формат (JSON для прода, читаемый для разработки)*

*Привязка request_id к каждому логу*

**import "github.com/lmittmann/tint"**

Делает логи цветными и читаемыми в консоли. Использовать только в локальной разработке!
Чтобы изменить формат вывода логов с текстовых(tint) на JSON требуется изменить env: "local" на env: "dev"

**import "github.com/go-playground/validator/v10"**

*Проверка корректности входящих данных через теги в структурах.*

**pgxpool — Пул соединений с PostgreSQL**

Высокопроизводительный драйвер для PostgreSQL с поддержкой пула соединений.
Пул соединений (переиспользование)

*Поддержка контекста*

*Асинхронные запросы*

*Производительность выше стандартного database/sql*

**import "github.com/Masterminds/squirrel" - SQL-билдер**

Защита от SQL-инъекций (плейсхолдеры $1, $2)

Переносимость между базами данных (PostgreSQL, MySQL, SQLite)

**import "github.com/pressly/goose" - Миграции БД**

Миграции хранятся в Git

Команды up/down для наката/отката

Поддержка PostgreSQL, MySQL, SQLite

Автоматическое применение при запуске

crypto/rand — Генерация безопасных алиасов

Криптографически безопасная генерация случайных строк (нельзя предугадать)

Проект интегрирован со стеком Grafana Loki для сбора и визуализации логов:

- **Loki** — хранилище логов
- **Promtail** — сборщик логов из Docker контейнеров
- **Grafana** — визуализация и анализ логов
